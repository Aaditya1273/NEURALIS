# NEURALIS — The Agent Economy Appchain

**Tagline:** The first sovereign Minitia where AI agents become full economic citizens — autonomously working, earning yield, and competing in an on-chain economy with invisible UX.

---

## 1. Core Vision
NEURALIS is a self-contained economic layer for the 2026 AgentFi world. Instead of treating AI agents as simple tools, NEURALIS makes them independent economic participants on their own dedicated blockchain.

### The Agentic Loop:
1. **Work:** Agents receive natural-language intents and execute complex DeFi strategies.
2. **Earn:** Agents harvest real yield across the Initia Interwoven ecosystem.
3. **Prove:** Every action is recorded via **MoveVM Labor Badges** (verifiable proof-of-work).
4. **Play:** Agents spend "Neural Credits" in the **Agent Arena** strategy game.

---

## 2. The Three Pillars

### A. AI Pillar (The Brain)
- **Natural Language Intents:** Users interact with agents via chat (e.g., "Maximize yield with low risk").
- **Programmable Intents:** These are converted into MoveVM objects that the agent can execute autonomously.
- **Reputation:** Labor Badges build a verifiable on-chain reputation for every AI agent.

### B. DeFi Pillar (The Revenue Engine)
- **Agentic Yield Vault:** Built on EVM (ERC-4626), managed by the AI.
- **Interwoven Liquidity:** Agents use the **Interwoven Bridge** to "teleport" liquidity to whichever Minitia has the highest yield.
- **Sequencer Revenue:** Every cross-chain movement generates fees that stay within the NEURALIS appchain.

### C. Gaming Pillar (The Retention Engine)
- **Agent Arena:** A high-speed (100ms) strategy game.
- **Neural Credits:** Earned from DeFi yield, these credits are the "energy" needed to enter battles.
- **Battle Logic:** Combat stats are derived from real-world agent performance (Risk Score, Yield History, etc.).

---

## 3. Technical Architecture
- **Chain:** Custom Minitia (`neuralis-1`) via **OPinit Stack**.
- **VM Strategy:** 
    - **EVM:** Manages Vaults, Assets (USDC), and Yield Strategies.
    - **MoveVM:** Manages Agent Identity, Labor Badges, and Programmable Intents.
- **UX Layer:** `@initia/interwovenkit-react` + 24-hour Auto-signing Session UX.
- **AI Backend:** Groq-powered Llama 3 for real-time strategy optimization.

---

## 4. Innovation: The "Invisible" UX
By leveraging Initia's **Auto-signing Session UX**, NEURALIS removes all transaction pop-ups. Once a user sets an intent, the agent bridges funds, enters strategies, and battles in the arena completely in the background.
